# Tuckys Gastos

Mini app para cargar gastos en el Google Sheet `Proyeccion Tuckys 2026`.

## Como funciona

- La persona carga fecha, importe, mes, categoria, concepto y descripcion.
- El sistema suma el importe en la celda correspondiente de `Cash-25 Tuckys`.
- Tambien guarda cada movimiento en una hoja de historial llamada `Carga Gastos Web`.
